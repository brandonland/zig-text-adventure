# zig-text-adventure
A text adventure written in Zig for obvious reasons

### This is a work in progress.

## Build
Dependencies:
- [Zig](https://ziglang.org/download/)

1. Clone this repo
2. `cd` into it
3. run `zig build run`
4. That's it :)
